USAGE:

Run Server With hot module reloading.
```
webpack-dev-server --hot --inline
```
or
```
npm start
```

Package files
```
webpack
```

Save NPM Dependencies
```
npm shrinkwrap
```
creates npm-shrinkwrap.json file.
https://docs.npmjs.com/cli/shrinkwrap
