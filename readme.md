# React Boilerplate

keep it simple stupid

## USAGE:

Install the deps the first time

```sh
npm i
```

Run the Dev Server

```sh
npm start
```

## Build:

```sh
npm run build
```

## Testing

```sh
npm test
```

## Docker

```sh
docker build -t react-boilerplate-image .
docker run -dit --rm --name react-boilerplate-app -p 8080:80 react-boilerplate-image
```
